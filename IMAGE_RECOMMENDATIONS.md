# Ocean Energy Image Recommendations

This file maps the live-site image contexts from `https://oenergy.co.uk/` to the placeholder filenames used in this repo.

## Visual Direction

- Keep the site industrial, technical, and credible.
- Prefer real process plants, real equipment, real engineers, and real safety context.
- Use wide shots for hero sections and tighter operational shots for service cards.
- Avoid generic business handshakes, lifestyle coworking imagery, random city skylines, and low-resolution collages.
- For dark hero sections, choose images that can survive a navy overlay without losing structure.

## If You Keep The Current Live Bootstrap Layout

The current live home page uses very small image slots for the seven service boxes. Those should not use stock photos.

Use clean SVG illustrations instead:

- Process: flowsheet or blueprint icon
- Technical Safety: shield + detector / flame sensor icon
- Control & Instrumentation: console / panel / gauge icon
- Piping & Plot Plan: pipe rack / elbow / valve icon
- Mechanical Static: vessel / tank icon
- Mechanical Rotating: compressor / pump icon
- Pipeline: long-route pipeline icon

That keeps the homepage readable while the larger service and hero areas use full photography.
